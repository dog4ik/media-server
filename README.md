# Media server / Library manager

### Key Features

- [x] Metadata fetching
- [x] Torrent client
- [x] Media transcoding

### Required dependencies

- `ffmpeg` and `ffprobe` are required.

### Supported metadata providers

- [TMDB](https://www.themoviedb.org/)
- [TVDB](https://thetvdb.com/)

### Supported torrent indexes

- TPB
